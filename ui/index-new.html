How It Works

Uses Signal Protocol (X3DH + Double Ratchet) with AES-256-GCM encryption

1. Each user generates identity key pair (ECDH P-256) 2. Session established via X3DH (shared secret, no server) 3. Every message advances Double Ratchet (per-message keys) 4. Relay server stores encrypted blob only 5. Recipient decrypts with their private key Result: Forward secrecy + post-compromise security
