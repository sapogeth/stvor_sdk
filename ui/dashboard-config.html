🎯 STVOR Dashboard Configuration

Проблема: Вы смотрите на онлайн дашборд https://sdk.stvor.xyz/dashboard.html, который настроен для продакшен сервера, а ваш relay работает локально.

✅ Вариант 1: Локальный дашборд (Рекомендуется)

Используйте встроенный дашборд вашего сервера:

http://localhost:3001/dashboard.html

Этот дашборд уже правильно настроен для вашего локального relay сервера.

🌐 Вариант 2: Туннель для внешнего доступа

Сделайте локальный сервер доступным из интернета:

# Установите ngrok
brew install ngrok

# Создайте туннель
ngrok http 3002

Затем используйте полученный URL в онлайн дашборде.

⚙️ Вариант 3: Настройка онлайн дашборда

Измените URL relay в настройках онлайн дашборда:

  1. Откройте https://sdk.stvor.xyz/dashboard.html
  2. Откройте Developer Tools (F12)
  3. В консоли выполните:
localStorage.setItem('stvor_relay_url', 'http://localhost:3002');
location.reload();
🔍 Статус вашего сервера:
• API Server: ✅ http://localhost:3001
• HTTP Relay: ✅ http://localhost:3002
• WebSocket: ✅ ws://localhost:8080
🚀 Открыть локальный дашборд
