🎯 Как это работает

Ваш SDK уже автоматически отправляет аналитику!

📊 Что отслеживается:

  • Шифрование сообщений - каждый раз при вызове client.send()
  • Расшифровка сообщений - при получении и успешной расшифровке
  • Отклоненные сообщения - при ошибках проверки подписи
  • Размер сообщений - для аналитики трафика
  • Версии SDK - для статистики использования

🔒 Безопасность:

  • Отправляются только метаданные (количество, размеры)
  • Содержимое сообщений НЕ отправляется
  • Пользователи анонимны (только хешированные ID)
  • Все метрики подписаны HMAC для защиты от подделки

🚀 Интеграция (Автоматически работает):

// Ваш код - аналитика работает автоматически! const client = await stvor.connect('user123'); await client.send('user456', 'Hello!'); // 📊 +1 encrypted message // Метрики отправляются на ваш сервер: // POST /analytics/event { event: 'message_encrypted', ... }
