Getting Started with STVOR

Install the SDK, connect users, and send encrypted messages in 3 simple steps.

1

Install the SDK

Add STVOR to your project with npm or yarn.

npm install @stvor/sdk

Zero dependencies — STVOR uses only native Web Crypto API

2

Connect a User

Initialize the client with your API credentials.

import { Stvor } from '@stvor/sdk'; const alice = await Stvor.connect({ userId: 'alice@example.com', appToken: 'stvor_live_xxx', relayUrl: 'https://relay.stvor.xyz', });
3

Send Encrypted Messages

Send messages to any peer — they're end-to-end encrypted automatically.

// Listen for incoming messages alice.onMessage(msg => { console.log(`From ${msg.sender}: ${msg.data}`); }); // Send an encrypted message await alice.send('bob@example.com', 'Hello encrypted!');

Common Use Cases

  • Chat Applications: Build encrypted messaging with zero server access to content
  • Financial Transactions: Secure peer-to-peer payments and fund transfers
  • Healthcare: HIPAA-compliant patient communication and data exchange
  • Legal Services: Attorney-client confidential communications
  • IoT & Control Systems: Secure device-to-device messaging

Security Guarantees

STVOR provides:

  • End-to-End Encryption: Only sender and recipient can read messages
  • Forward Secrecy: Past messages are safe even if keys are compromised
  • Post-Compromise Security: Future messages are secure after key compromise
  • Authenticated Encryption: Messages are cryptographically authenticated
  • Zero Knowledge: Server never sees message content
