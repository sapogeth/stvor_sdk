Why STVOR

Everything you need.
Nothing you don't.

Built on the same cryptography as Signal — without the complexity.

🔐

Signal Protocol

X3DH key agreement + Double Ratchet. The same cryptography used by Signal, WhatsApp, and Wire.

⚡

Zero dependencies

Node.js SDK uses only built-in node:crypto. Browser SDK uses Web Crypto API. No WASM, no npm bloat.

🔄

Forward Secrecy

DH ratchet rotates keys every message. Compromise of today's keys exposes nothing from the past.

🌐

Browser support

Full Double Ratchet in the browser via Web Crypto API. Same API, same security — import from @stvor/sdk/web.

📦

Any data type

Send strings, numbers, objects, Dates, Sets, Maps, Uint8Arrays. Exact type is preserved on delivery.

🛡️

Replay protection

Nonce + timestamp validation on every message. Messages are deleted from relay after delivery.

How it works

Simple API, serious crypto.

Three steps. No cryptography expertise required.

1

Install & start mock relay

npm install @stvor/sdk then npx @stvor/sdk mock-relay — local relay on port 4444, no account needed.

2

Connect a user

Stvor.connect({ userId, appToken, relayUrl }) — generates identity keys, registers with relay, starts polling. One call, fully automatic.

3

Send & receive

alice.send('bob', data) — X3DH + Double Ratchet happens automatically. Recipient gets the message via onMessage, type preserved.

# Stvor SDK
End-to-end encryption for any app. Signal Protocol (X3DH + Double Ratchet). Zero runtime dependencies.
Install
npm install @stvor/sdk
Quickstart
import { Stvor } from '@stvor/sdk'; const alice = await Stvor.connect({ userId: 'alice', appToken: 'stvor_live_xxx', relayUrl: 'https://relay.stvor.xyz', }); const bob = await Stvor.connect({ userId: 'bob', appToken: 'stvor_live_xxx', relayUrl: 'https://relay.stvor.xyz', }); bob.onMessage(msg => console.log(msg.from, msg.data)); await alice.send('bob', { text: 'Hello!' }); await alice.disconnect(); await bob.disconnect();
Stvor.connect(config)
userIdstring — any unique identifier
appTokenstring — must start with 'stvor_'
relayUrlstring — 'https://relay.stvor.xyz' or self-hosted
timeout?number — ms, default: 10000
pollIntervalMs?number — polling interval, default: 1000
client.send(recipientId, data, options?)
await client.send('bob', 'Hello'); await client.send('bob', { amount: 100, currency: 'USD' }); await client.send('bob', Buffer.from([1, 2, 3])); await client.send('bob', new Date()); await client.send('bob', 'Hey', { timeout: 30_000 }); await client.send('bob', 'Hey', { waitForRecipient: false });
client.onMessage(handler)
const unsub = client.onMessage(msg => { // msg.from — sender userId // msg.data — decrypted value (original type preserved) // msg.timestamp — Date // msg.id — unique string }); unsub(); // stop listening
